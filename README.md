## 📘 Course Description  
This course teaches the design and development of modern web applications using AI-assisted development tools and workflows. Students will learn to leverage AI coding assistants (GitHub Copilot, ChatGPT, Claude) to build full-stack applications while developing critical skills in prompt engineering, evaluating AI-generated code, and debugging complex systems. Topics include:
- AI-assisted web development and prompt engineering  
- React front-end and Node.js/.NET back-end development  
- RESTful API design and data persistence  
- Authentication, security, and OAuth/OpenID Connect  
- CI/CD pipelines and cloud deployment  
- Reading, navigating, and debugging large AI-generated codebases  
- Testing strategies for AI-assisted workflows  

---

## 🎯 Learning Outcomes  
Students will:
- Build modern full-stack web applications using AI-assisted development tools and workflows  
- Write effective prompts to generate, refactor, and debug code using AI coding assistants  
- Critically evaluate, understand, and explain AI-generated code before integrating it into projects  
- Implement secure authentication and authorization using OAuth/OpenID Connect  
- Design and consume RESTful APIs with proper data persistence  
- Set up CI/CD pipelines and deploy applications to cloud platforms  
- Apply testing strategies to validate AI-assisted codebases  
- Navigate, debug, and maintain large codebases produced through AI-assisted development

## 📝 Homework Format  
Homework assignments are designed to build practical AI-assisted development skills. Each assignment may include:
- **Building with AI:** Using AI tools to scaffold, implement, and iterate on web application features  
- **Debugging Broken Codebases:** Diagnosing and fixing intentionally broken projects to sharpen debugging skills  
- **Prompt Documentation:** Maintaining a prompt log that records your AI interactions, including the prompts used, outputs received, and any modifications made  

All homework submissions must include your prompt log alongside your code.
Use [`resources/prompt-log-template.md`](resources/prompt-log-template.md) as the default format unless a specific assignment says otherwise.

## 🏗️ Final Project  
The final project is a full-stack web application that demonstrates mastery of the course topics. Requirements include:
- A complete front-end and back-end with user authentication via **OAuth/OpenID Connect**  
- A **CI/CD pipeline** using GitHub Actions  
- Deployment to **Microsoft Azure**  
- A comprehensive **prompt log** documenting all AI-assisted development  
- A **security review** identifying and addressing potential vulnerabilities  

See [`week10/final-project-spec.md`](week10/final-project-spec.md) for the full specification, [`resources/prompt-log-template.md`](resources/prompt-log-template.md) for the required prompt log format, and [`resources/security-review-template.md`](resources/security-review-template.md) for the security review checklist.

### 🤖 AI Usage Policy
AI tools (GitHub Copilot, ChatGPT, Claude) are **encouraged** and expected in this course. The following guidelines apply:
- **Document all AI usage:** All AI interactions must be recorded in a prompt log using the provided template. This includes prompts, outputs, and any modifications you made to the generated code.  
- **Understand what you submit:** You must be able to explain every line of code in your submissions. Instructors may ask you to walk through your code during class or office hours.  
- **AI-generated code without understanding is a violation:** Submitting code you cannot explain — even if it works — is considered an academic integrity violation.  
- **No copying from classmates:** Direct copying of another student's prompts, prompt logs, or code is prohibited and subject to academic integrity sanctions.  
- **Iterate and learn:** The goal is to use AI as a learning accelerator, not a replacement for understanding. Your prompt log should show growth and iteration over time.
